# KlipNote UI 设计交接文档

## 项目概述
KlipNote 是一个AI音频转录工具，支持多模型转录、实时编辑和多格式导出。

## 目标用户
- 内容创作者（播客、视频制作者）
- 研究人员（访谈记录、会议纪要）
- 媒体工作者（字幕制作）

## 核心用户场景
1. **快速转录**：上传音频 → 选择模型 → 自动转录
2. **精细编辑**：查看转录结果 → 点击时间戳跳转 → 编辑文本
3. **导出分享**：快速导出TXT/SRT，或高级导出其他格式

## 设计要点
- **响应式设计**：适配桌面和移动端
- **深色/浅色模式**：参考图片为深色模式
- **时间戳精度**：毫秒级（00:00.12格式）
- **双重导出**：快速按钮 + 完整对话框

💬 设计要点

1. 设计目标对话

"我们需要一个专业但不失亲和力的界面。用户可能会长时间使用编辑功能，所以界面要：
- 清晰易读（合适的字体大小和行距）
- 减少视觉疲劳（柔和的配色，充足的留白）
- 操作高效（快捷键支持，快速导出）"

2. 功能优先级说明

核心功能（必须突出）：
✓ 音频播放控制
✓ 转录文本编辑区
✓ 时间戳跳转
✓ 快速导出TXT/SRT

次要功能（可以收起）：
- 说话人管理
- 质量指标
- 高级导出选项

3. 参考图的使用方式

"参考图（transcript_result_1.png）展示了我们期望的：
- 时间戳显示方式（00:00.12 - 00:05.87 格式）
- 深色主题风格
- TXT/SRT快速导出按钮位置

但请自由发挥，这只是参考，不是严格复制。你可以：
- 调整配色方案（提供浅色/深色两套）
- 优化按钮样式和图标设计
- 改进布局和间距
- 添加悬浮提示等交互细节"

4. 技术约束和规范

- 使用现代Web技术（Vue 3 + TypeScript）
- 设计规范：建议8px网格系统
- 字体：建议使用系统字体栈或思源黑体
- 图标：可以使用Heroicons或Material Icons
- 响应式断点：移动端(<768px)、平板(768-1024px)、桌面(>1024px)
